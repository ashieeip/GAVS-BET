# Baccarat Betting Game (Browser)

A simple Baccarat betting game built with **HTML/CSS/JavaScript**.

## Run
1. Open `index.html` in your browser.
2. Choose a **wager**.
3. Click **Bet Player** or **Bet Banker**.
4. Click **Deal**.

## Notes
- This is a simplified payout model (Player/Banker win pays **1:1**, ties return the wager).
- Card totals use standard Baccarat rules (mod 10; face cards count as 0).
